# chemistry-software-htmx-go
A simple web app using Go, HTMX and PicoCSS

## Installation

### Windows

1. Install Go from the official website: https://go.dev/dl/
2. Open a command prompt and run `go get -u github.com/chemistry-software/chemistry-software-htmx-go`
3. Run `go run main.go`

### Linux

1. Install Go from the official website: https://go.dev/dl/
2. Open a terminal and run `go get -u github.com/chemistry-software/chemistry-software-htmx-go`
3. Run `go run main.go`

## Usage

After running the app, navigate to `http://localhost:8080` in your web browser.
